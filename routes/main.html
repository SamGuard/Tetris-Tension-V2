Welcome to Tetris Tension.


This is Tetris, but with a twist. You and a partner will be playing Tetris, however one of you will be able to see the game board but not control the game, while the other will be able to control the falling pieces but not see the board. You work together as a team to get the highest score possible.

Starting a game:
To play the game, you need to first find someone you dont want to be friends with anymore. Once you have done this, one player should click on "Create Game", where you will be given a 6 digit game pin. The other player should type this pin in and click on "Join Game". This takes both players to the waiting screen where the person who created the game clicks the "Ready" button to start the game.

Scoring:
8 points for clearing a row, this is increased exponentially for every consecutive row cleared, using the formula:
New Score = Old Score + 8^(Number of rows cleared)
So if you manage to clear 3 rows in 1 go you will get 8^3 added to your score.
